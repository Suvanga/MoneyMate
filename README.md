# MoneyMate 💸

MoneyMate is an intelligent, full-stack personal finance and debt-tracking application designed to help users manage expenses, track debts, and scan bills effortlessly. Built with a React Native frontend and a robust Node.js/Express backend, it leverages AI and OCR technology to automate financial logging.

**Presentation & Demo:** [View the Project Presentation from the repo

## 🚀 Tech Stack

* **Frontend:** React Native, Expo, Axios, React Navigation
* **Backend:** Node.js, Express.js, MongoDB (Mongoose)
* **Authentication:** Auth0
* **AI & Utilities:** OpenAI API (for insights/feedback), OCR integration

## ✨ Key Features

* **Secure Authentication:** Seamless user login and registration powered by Auth0.
* **Smart Bill Scanning:** Use your device's camera to capture receipts and extract data via OCR.
* **Debt Management:** Track who owes you, who you owe, and settle up with an intuitive dashboard.
* **AI Financial Insights:** Get intelligent feedback on your spending habits using OpenAI integration.

## 💻 How to Run Locally (Mac Preferred)

### Prerequisites
* **Node.js** installed on your machine.
* **Xcode** installed with the latest iOS Simulator.
* **Expo CLI** installed globally (`npm install -g expo-cli`).
